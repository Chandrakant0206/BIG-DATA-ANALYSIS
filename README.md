# Big Data Analysis Using PySpark

## Internship Task

This project is part of the **CODTECH IT Solutions Data Analytics Internship**.

## Objective

The goal of this task is to perform **Big Data Analysis** using **PySpark** on a large dataset and extract useful insights.

## Dataset

Netflix Movies and TV Shows Dataset

The dataset contains information such as:

* Title
* Director
* Cast
* Country
* Release Year
* Rating
* Duration
* Genre

## Tools & Technologies

* Python
* PySpark
* Google Colab
* GitHub

## Data Preprocessing

The following preprocessing steps were performed:

* Checked missing values
* Removed null values
* Removed duplicate records
* Converted date column to proper format

## Data Analysis

The following analysis was performed:

1. Movies vs TV Shows distribution
2. Top countries producing Netflix content
3. Content released per year
4. Most common ratings
5. Most popular genres

## Insights

* Movies are more common than TV Shows on Netflix.
* United States produces the highest number of titles.
* Content production increased significantly after 2015.
* Drama and International Movies are among the most common genres.

## Conclusion

This project demonstrates how **PySpark can efficiently process and analyze large datasets**, making it suitable for big data analytics tasks.
